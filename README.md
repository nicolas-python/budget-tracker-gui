# budget-tracker-gui
Ein Python-basierter Budget-Tracker mit einer grafischen Benutzeroberfläche.
Erlaubt das Eintragen von Einnahmen und Ausgaben, das Kategorisieren und das Anzeigen der Werte.

## Funktionen
 Eingabefelder für:
  - **Typ** (Einnahme / Ausgabe)
  - **Kategorie** (z. B. Fixkosten, Hobbys,Essen)
  - **Preis**
